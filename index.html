What is this about?

This project is my own take on creating particle effects. Some examples of particles include rain, snow, sand, electrons as well as atoms, confetti and volcano eruption debris, but gasses or fluids could also be approximated by particles.

Particle effects are a computer simulation of particles aimed, usually, at making something look beliveable and/or nice. They are mainly used in video games and movies, to create sparks, show dust raising after a player runs across it, portray a magic spell, and many more similar effects.

Project structure

Particles are drawn on HTML5 <canvas> elements using Javascript. Each particle effect is handled by a separate class that extends a shared, base ParticleEnv class.

The code in the shared ParticleEnv class sets up the canvas, frames and updates the width and height of the canvas to match its container.

The indivdual classes handle creating particles and updating them every frame. Rain and snow, showcased above, also use a shared WindSystem class that handles wind direction and strength.
